# Animated icons

Icons is located in the folder `web/themes/custom/hoeringsportal/templates/animated-svg`

Corresponding css is locatend in `web/themes/custom/hoeringsportal/assets/css/module/_animated-svg.scss`.


## Location icon

Example of use in a twig file:

``` html
<div style="color: var(--bs-pink);">
  {{ include(directory ~'/templates/animated-svg/icon-deltag-location.svg.html.twig') }}
</div>
```
